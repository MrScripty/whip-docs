Vulkan Engine Module Map

This diagram visualizes the relationships between modules in the Rust-based Vulkan engine project. Nodes represent files, and links represent dependencies or communication paths.

Controls:

  • Click a node to select/deselect it.
  • Use the middle mouse button scroll wheel to zoom in/out.
  • Press and drag the middle mouse button to pan the graph.
  • Drag individual nodes with the left mouse button.
